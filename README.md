# Fintech-Task-Manager & Notification Backend

Backend project inspired by OneFin's backend developer JD.

## Tech Stack

- Python
- Django & Django REST Framework
- FastAPI
- PostgreSQL
- Celery & Redis
- Docker (later)
- Pytest

## Features

- Task management APIs (CRUD)
- Client management APIs
- Background job processing for notifications using Celery + Redis
- Analytics / health microservice using FastAPI
- Token-based authentication
